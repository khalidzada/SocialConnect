1. Introduction

Social Connect+ is a social media and e-commerce hybrid platform integrating user interactions, real-time messaging, and online shopping.

2. System Architecture

2.1 High-Level Overview

The system follows a Microservices Architecture with key components such as .NET Core Web API, React.js frontend, and SQL Server.

2.2 Component Breakdown

  • User Management (Authentication, Profile, Follow/Unfollow)
  • Social Media Module (Post creation, Likes, Comments)
  • Real-Time Chat & Notifications (SignalR)
  • E-commerce Module (Shop & Product Management, Payments)

3. Database Design

3.1 Entity Relationship Diagram (ERD)

[User]----(1:N)----[Post]----(1:N)----[Comment]
[User]----(1:N)----[Like]
[User]----(1:N)----[Message]
[User]----(1:N)----[Notification]
[User]----(1:N)----[Shop]----(1:N)----[Product]

3.2 Table Schemas

Users Table

Column Type Constraints
UserId INT (PK) Auto Increment
Username VARCHAR(50) UNIQUE, NOT NULL

4. API Specifications

4.1 Authentication APIs

  • POST /api/auth/register
  • POST /api/auth/login
  • GET /api/auth/profile/{id}

5. Technology Stack

Component Technology
Frontend React.js / Next.js
Backend .NET Core Web API
Database SQL Server

6. Deployment Strategy

CI/CD is handled using GitHub Actions, and hosting is on Azure/AWS.

7. Security Considerations

  • JWT Authentication
  • SQL Injection Prevention
  • Data Encryption for Sensitive Information

8. Conclusion

This document provides a structured technical foundation for Social Connect+, ensuring modularity, scalability, and security.
